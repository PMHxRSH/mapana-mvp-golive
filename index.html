Project Charter Summary

Problem Statement

Nutrient analysis is vital for crop growth, efficient fertilizer use, stable yields, and sustainable farming. Traditional methods are slow and imprecise, creating supply chain risks. Mapana solves this with an integrated solution using spectrometry, machine learning, and expert agronomy. Unlike existing tools that rely on secondary datasets, Mapana measures nutrients directly from crops, delivering real-time insights. This enables corporations to secure supply chains, scale globally, and improve competitiveness, while also building long-term nutritional datasets for machine learning.

With proactive crop nutrition management, organizations gain real-time analysis, systematic reporting, and actionable insights — leading to better yields, cost savings, reduced risks, and more sustainable farming.

Project Objectives

  1. Develop a mobile application that connects with the Mapana hardware device and integrates the entire value chain from sample collection to analysis to nutrition application and monitoring.
  2. Build a system to support lab quality control.
  3. Create a reliable system to manage and maintain the full Chain of Custody.
  4. Develop a dashboard system.
The above project objectives are met via 4 systems, i.e.:
  1. Field User Mobile Application — Sample Chain of custody, NAP Application confirmation, Post NAP growth recording. (Pulse)
  2. A Tablet Application to pair with the Mapana Blackbox. (Prism)
  3. A Web Based Application (for Configuration). (Scope)
  4. A Dashboard. (Scope + Signal ML APIs)
